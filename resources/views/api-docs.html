API Documentation

Integrate PDF Creator's powerful conversion engine into your own applications.

1 Authentication

All API requests must include the X-API-Key header.

Header: X-API-Key: {{ api_key }}

2 Base URL

https://pdfcreator.rosyappsstudio.com/api

Endpoints

POST

/convert

Unified Conversion

The primary endpoint for all file-based conversions. Automatically detects input format.

Form Data Parameters

file UploadFile (Binary) - Required
target_format String (e.g., "pdf") - Default: "pdf"
filename String (Optional)
POST

/html-to-pdf/code

HTML Code to PDF

Converts raw HTML string content to a PDF document.

JSON Body

{ "html_content": "<h1>Hello World</h1>", "filename": "output.pdf" }
POST

/html-to-pdf/link

Web URL to PDF

Fetches a public URL and renders it as a PDF.

Form Data Parameters

url String (Required)

Response Format

All successful requests return a JSON object with the file URL.

{ "status": "success", "file_url": "https://pdfcreator.rosyappsstudio.com/storage/converted/doc_12345.pdf" }
